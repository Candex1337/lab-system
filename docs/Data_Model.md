# Выбор стека технологий

В качестве основного языка программирования используется **JavaScript**. 
Данный язык позволяет разрабатывать как клиентскую, так и серверную часть приложения, что упрощает разработку и сопровождение системы.

В качестве среды проектирования и разработки используется **Visual Studio Code**. 
Данная среда обладает удобным интерфейсом, поддерживает работу с современными веб-технологиями, расширениями и системой контроля версий Git.

Для серверной части приложения выбран **Node.js**. 
Использование Node.js позволяет разрабатывать серверную логику на языке JavaScript и обеспечивает удобную работу с сетевыми запросами и API.

В качестве фреймворка выбран **Express.js**. 
Express.js позволяет быстро создавать веб-приложения, настраивать маршруты, обрабатывать запросы пользователей и реализовывать серверную бизнес-логику.

В качестве системы управления базами данных выбрана **MongoDB**. 
MongoDB является документо-ориентированной СУБД, что удобно для хранения данных о пациентах, анализах, результатах исследований и протоколах. 
Использование MongoDB позволяет гибко хранить и изменять структуру данных в процессе разработки проекта.
