# **Выбор темы проекта и обоснование модели проектирования**

## **Тема проекта:** Система управления медицинской лабораторией.

Разрабатываемый проект представляет собой информационную систему, предназначенную для автоматизации работы медицинской лаборатории. Система обеспечивает регистрацию пациентов, назначение анализов, внесение результатов исследований, формирование протоколов и печать бланков.

Для разработки проекта выбрана модель Agile. Данная модель позволяет разрабатывать систему поэтапно, реализуя отдельные модули и постепенно расширяя функциональность.

Выбор модели Agile обусловлен тем, что проект состоит из нескольких независимых частей (модулей), которые удобно разрабатывать и тестировать поэтапно.
