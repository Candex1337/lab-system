# Система управления медицинской лабораторией

## Описание проекта
Информационная система для автоматизации работы медицинской лаборатории.

## Основные функции
- регистрация пациентов
- назначение анализов
- сбор результатов
- формирование протоколов
- печать бланков

## Цель проекта
Автоматизация работы лаборатории, уменьшение ошибок при обработке данных и ускорение подготовки результатов.
